## What is the difference between var, let, and const?
=> var: It's a keyword that we use to declare variable in javascript. It is function-scoped variable. We can redeclare it and reassign it's value.

let: It's a keyword to declare a block-scoped variable in javascript. It was introduced in ES6. We can't redaclare it but reassign it's value.

const: It's also a block-scoped variable. It's constant so we can't reassign it's value after initialization.

## What is the spread operator (...)?
=> This operator is used to expand or spread the elements of an array. We can expand an object too into individual elements. With it's help we can copy an array, merge or pass the values.

## What is the difference between map(), filter(), and forEach()?
=> map(): map is a method which is used to create a new array by applying a function to each element of the original array.

filter(): filter is a method which is used to create a new array but it only contains the elements which matches a specific condition.

forEach(): This array method  is almost like map method, it also applies a function to each element of an array but it does not return a new array.

## What is an arrow function?
=> Arrow function is a shorter syntax of writing functions in javascript. It was intoduced in ES6. It uses => symbol to declare a function.

## What are template literals?
=> Template literals are a great feature in javascript that is used to create strings more easily using backticks (``) istead of single or double quotes. In that backticks we can add variables or any expressions inside a string using ${} and we can write multi line strings, fun fact we can even call a function in it and create a html file in it too.
